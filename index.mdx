Tienda de Playeras is a Service-Oriented Architecture (SOA) project built as part of a university course on service-oriented systems. It consists of two independent components: a **Java/Jakarta EE 10 REST backend** deployed on Apache Tomcat and a **Laravel 12 PHP frontend** that communicates with it over HTTP. Together they provide a complete product catalog with full CRUD operations and real-time inventory management.

## Quick Start

Get both services running locally in four steps.

<Steps>
  <Step title="Set up the database">
    Create a MySQL database named `tienda_playeras` and add a `productos` table with columns `id`, `nombre`, `talla`, `precio`, and `stock`.
  </Step>
  <Step title="Build and deploy the Java server">
    Build `ServidorTiendaPlayeras` with Maven (`mvn package`) and deploy the generated WAR to Apache Tomcat on port `8080`.
  </Step>
  <Step title="Configure the Laravel client">
    Copy `.env.example` to `.env`, set your `APP_KEY`, and run `composer install` followed by `php artisan serve` in the `ClienteTiendaPlayeras` directory.
  </Step>
  <Step title="Open the store">
    Navigate to `http://localhost:8000` to see the product catalog. Add, edit, delete, and sell t-shirts from the dashboard.
  </Step>
</Steps>

## Key Features

    Create, read, update, and delete t-shirt products with name, size, price, and stock fields.
  </Card>
  <Card title="Inventory Management" icon="warehouse">
    Atomic stock decrement on every sale prevents overselling even under concurrent requests.

    Responsive product card grid with per-product edit modals and quantity-based sell forms.
